# Leomayn Website

Next.js website for Leomayn Solutions Ltd - AI operations and automation consultancy.

## Tech Stack

- **Framework:** Next.js 14+ with App Router
- **Styling:** Tailwind CSS
- **Language:** TypeScript
- **Hosting:** TBD (Vercel recommended)

## Project Structure

```
leomayn-website/
├── src/
│   ├── app/              # Next.js app router pages
│   ├── components/       # React components
│   └── styles/           # Global styles
├── public/               # Static assets
├── docs-git-ignore/      # Private docs (not in Git)
└── README.md             # This file
```

## Private Documentation

Implementation briefs and working files are in `docs-git-ignore/` (excluded from Git).

## Development

```bash
# Install dependencies
npm install

# Run development server
npm run dev

# Build for production
npm run build
```
